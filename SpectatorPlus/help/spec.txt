§6§lThis command is the base command of SpectatorPlus.
§rUse it to change the spectator mode of the players, to create arenas, to change options of the plugin...
on:Enables the spectator mode for the given player (or yourself).
off:Disables the spectator mode for the given player (or yourself).
say:Broadcasts a message to the spectators.
togglehide:Hides a player from the spectators, even if the mode allows him to be seen (excepted for the players with a sufficient permission).
list:Lists all the spectators in this server.
mode:Changes the players a spectator can see in the teleporter.
arena:Manages arenas: creation, deletion, list, lobby...
lobby:Sets or deletes the spectators lobby, where the players are teleported when they enter or leave the spectator mode (if set).
config:Updates the config from the game, with instant application of the new rules.
b:A workaround command to exit the no-clip mode, because some versions of CraftBukkit 1.8 don't allow plugins to listen to clicks for spectators.
help:Displays this help.
